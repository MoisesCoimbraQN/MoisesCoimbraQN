### 📁 Principais Projetos

#### 🚀 [SafeAsset](https://github.com/moisescoimbra/safeasset)
Solução de dados desenvolvida individualmente para o Challenge NUCLEA. O projeto engloba desde a ingestão e tratamento de dados brutos até a aplicação de modelos preditivos e regras de negócio para mitigação de riscos e segurança de ativos financeiros. 
* *Tecnologias:* Python, Machine Learning, SQL, Engenharia de Pipelines.

#### [Preservar IA e Crédito](https://github.com/moisescoimbra/Preservar-IA-e-Credito)
Solução inteligente de concessão de crédito rural desenvolvida para o Desafio Nuclea, integrando Ciência de Dados e Sensoriamento Remoto para precificação de risco socioambiental (ESG). O projeto automatiza a esteira de crédito do produtor a partir de dados históricos de satélite e regras do Código Florestal, classificando imóveis em alçadas com arquitetura resiliente.

Tecnologias: Python, XGBoost, Google Earth Engine, Dash (Plotly), GeoJSON, Render.
